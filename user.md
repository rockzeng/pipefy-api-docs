## GET /users/:id.json

Gets the JSON representation of a [user]("user.md") of pipefy.

```json
{
  "id": 1,
  "name": "John",
  "username": "john",
  "email": "john@mail.com",
  "created_at": "2014-01-01T16:18:03.602-03:00",
  "updated_at": "2015-01-01T10:49:05.288-03:00",
  "display_username": "john"
}
```
